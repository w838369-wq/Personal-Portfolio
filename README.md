#Personal Portfolio

A personal portfolio website built with React + Vite, showcasing projects, skills, and background.

## Tech Stack

- **React 19** — UI framework
- **Vite 7** — build tool with HMR
- **JavaScript (ES Modules)**
- **CSS** — custom styles with responsive layout

## Project Structure

```
src/
├── components/       # Reusable UI components
│   ├── Hero.jsx
│   ├── Navigation.jsx
│   ├── Footer.jsx
│   ├── ProjectCard.jsx
│   ├── TabSection.jsx
│   └── ContactForm.jsx
├── pages/            # Page-level components
│   ├── Home.jsx
│   ├── About.jsx
│   └── Contact.jsx
├── data/
│   └── projectsData.js   # Project metadata
└── styles/           # CSS stylesheets
```

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
